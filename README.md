## 🛒 Online Store API

A production-ready E-Commerce REST API built with 
ASP.NET Core 8 and Clean Architecture.

### Features
- JWT Authentication + Refresh Tokens
- Role-based Authorization (Admin/Customer)
- Product Management with Image Upload
- Order, Payment & Shipping Management
- Rate Limiting & Pagination
- Global Error Handling
- Swagger Documentation

### Tech Stack
ASP.NET Core 8 | EF Core | SQL Server | JWT | BCrypt

### Architecture
Clean Architecture with 4 layers:
Store.Core → Store.Repository → Store.Service → Store.API
